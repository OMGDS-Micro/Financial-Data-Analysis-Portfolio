# 📊 Financial Data Analysis Portfolio | 金融數據分析作品集

### 👨‍💻 關於我 (About Me)
我是經濟系畢業的數據分析學習者，專注於利用 **Python** 解決金融實務問題。
本專案展示了如何運用資料科學工具（Pandas, NumPy, Matplotlib）進行**匯率趨勢分析**、**信貸風險控管**以及**自動化對帳**。

---

### 🚀 專案內容 (Projects)

#### 1. 多幣別匯率趨勢儀表板 (FX Trend Monitor)
* **目標**：解決人工抓取匯率效率低落的問題，輔助財務換匯決策。
* **技術**：計算 **MA30/MA7 移動平均線**，自動標記 **黃金交叉 (Golden Cross)** 作為買進訊號。
* **工具**：`Pandas` (時序處理), `Matplotlib` (金融視覺化)。

#### 2. 信貸還款壓力測試模擬器 (Loan Stress Tester)
* **目標**：模擬央行升息情境，計算客戶在不同利率下的**收支比 (DBR)** 與違約風險。
* **技術**：實作 **PMT 年金現值公式**，進行敏感度分析 (Sensitivity Analysis)。
* **工具**：`NumPy`, `For Loop` (情境模擬)。

#### 3. 自動化對帳機器人 (Reconciliation Bot)
* **目標**：解決銀行與企業內部帳務不符的痛點，自動抓出漏帳與溢帳。
* **技術**：使用 **Outer Join** 進行全盤比對，自動生成差異報表。
* **工具**：`Pandas Merge` (資料庫邏輯), `Logic Optimization`。
